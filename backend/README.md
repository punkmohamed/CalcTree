# Personal Coach API

A backend API for a personal coaching platform with e-commerce and course management capabilities, including video streaming, payments, and user management.

## Tech Stack

- **Node.js** with TypeScript
- **Express.js** - Web framework
- **MongoDB** with Mongoose - Database
- **JWT** - Authentication with session management
- **Vimeo API** - Video streaming
- **Stripe** - Payment processing
- **Cloudinary** - File storage
- **Nodemailer** - Email service
- **Zod** - Validation

## Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd Personal-Coach-Api
```

### Install Dependencies

```bash
npm install
```

### Setup Environment Variables

Create a `.env` file in the root directory and configure your environment variables.

### Run the Application

```bash
npm run dev
```

The API will start on `http://localhost:4000` (or your configured PORT).
